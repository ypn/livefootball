
======== BẢO KIM - THANH TOÁN THẺ CÀO =========

1. Thông tin kết nối bao gồm:
	- Mã website
	- Mật khẩu website
	- Tài khoản API Username
	- Tài khoản API Password

Nếu bạn chưa có những thông tin trên xin vui lòng truy cập vào địa chỉ https://www.baokim.vn/developers/tai-lieu/tich-hop-the-cao#info để xem hướng dẫn.

2. Tiến hành kết nối
	- Tại file transaction.php các bạn điền những thông tin sau:
		merchant_id : Mã website đăng ký
		secure_code : Mật khẩu website
		api_username : Tài khoản API Username
		api_password : Tài khoản API Password
	
Sau khi hoàn thành các bạn đã có thể tiến hành thanh toán sử dụng phương thức thanh toán qua thẻ cào của Bảo Kim.
